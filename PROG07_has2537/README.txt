This program module will run simple Binary search tree header program in three different sorted order:
post-order traversal, In-order traversal, pre-order traversal.

The header file is meant to run with a driver main file main.cpp.
The functions include inserting nodes, counting nodes, checking for empty nodes and deleting nodes.

*ALL COMPILATION AND RUN INSTRUCTIONS FOR LINUX TERMINAL WITH GCC COMPILER INSTALLED*

To compile:

- enter
  g++ -std=c++11 -o mybst main.cpp

  in linux terminal.

- or simply enter "make" with no arguments or quotation marks.

This will compile c program into an executable name UTPod

To run:

- Enter "./mybst <name_of_file>" without quotes in terminal. name_of_file will be the name of whatever file
represents the picture in the directory.
